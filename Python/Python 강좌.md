
6) 변수와 입력 사용하기

   - 영문 문자와 숫자를 사용할 수 있습니다.
   - 대소문자를 구분합니다.
   - 문자부터 시작해야 하며 숫자부터 시작하면 안 됩니다.
   - _(밑줄 문자)로 시작할 수 있습니다.
   - 특수 문자(+, -, *, /, $, @, &, % 등)는 사용할 수 없습니다.
   - 파이썬의 키워드(if, for, while, and, or 등)는 사용할 수 없습니다.

   ```python
   # input으로 값 입력 받기
   x = input('i에서 10사이의 정수값>')
   # split으로 값 여러개 받기
   a, b, c = 'a quick brown'.split()
   
   ```
